# 一、描述

用于构建个人博客



Github库：

https://github.com/wengshenggui/wengshenggui.github.io



# 二、项目

1、安装依赖：

```shell
bundle install
```

2、启动项目：

```shell
bundle exec jekyll serve
```

3、启动项目带预览草稿：

```shell
bundle exec jekyll serve --drafts
```
